# 🚗 Challenger Hellcat — Landing Page

Este projeto é uma **landing page inspirada no Dodge Challenger Hellcat**, desenvolvida com foco em **prática de front-end**. O objetivo principal foi treinar habilidades de estruturação, estilização e organização visual utilizando HTML e CSS (e possivelmente JavaScript, se aplicável).

> ⚠️ Este projeto é apenas um exercício educacional, sem fins comerciais.

---

## 🎯 Objetivo

Reproduzir uma interface moderna e impactante, trabalhando:

- Layouts responsivos
- Uso de **CSS Grid e/ou Flexbox**
- Tipografia e hierarquia visual
- Aplicação de cores e contraste
- Seções bem definidas (hero, specs, call to action, etc.)

---

## 🖼️ Preview

A landing page apresenta:

- **Hero section** com imagem de destaque e CTA
- Seção de **performance e especificações**
- Destaque para motor e potência
- Área visual do cockpit
- Grid com métricas (velocidade, torque, etc.)
- Seção final com **formulário de inscrição**

---

## 🛠️ Tecnologias Utilizadas

- HTML5
- CSS3
- (Opcional) JavaScript

---

## 📚 Aprendizados

Durante o desenvolvimento, foram praticados:

- Organização semântica do HTML
- Criação de layouts complexos
- Uso de espaçamento e alinhamento
- Estilização com foco em design moderno
- Estruturação de componentes visuais

---

## ⚠️ Observações Importantes

- Este projeto **não é oficial** e não possui vínculo com nenhuma marca.
- As imagens e referências visuais foram utilizadas apenas para fins de estudo.
- Não há backend ou funcionalidades reais de envio de dados.

---

## 🚀 Possíveis Melhorias

- Tornar totalmente responsivo para mobile
- Adicionar interações com JavaScript
- Implementar animações (hover, scroll, etc.)
- Criar versão com componentes reutilizáveis (ex: React)
